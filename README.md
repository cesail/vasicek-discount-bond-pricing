> Longer‑maturity risk‑free rates are built from the short rate r(t). The price of a zero coupon (discount) bond at time t is determined by the expected average path of r(t) between a time point t and maturity.

**In this notebook, we simulate the Vasicek short‑rate model for r(t) to price an n‑year zero coupon bond at any time t between issuance and maturity.**
